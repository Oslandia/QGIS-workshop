QT¶

Qu’est ce que c’est ?¶

Qt (prononcé : kioute soit “joli”) est un framework (ensemble d’outils) applicatif opensource développé par Nokia pour construire des interfaces graphiques utilisateur (GUIs) et des applications. Vous avez surement déjà travaillé avec des logiciels créés avec Qt si vous avez touché à ces programmes : Google Earth, Virtual Box, Skype, Autodesk, les smartphones Android… QGIS est aussi construit avec Qt, et c’est pour ça que nous en sommes là.

Le but d’utiliser un framework applicatif tel que Qt est qu’il aide à accélérer les cycles de développement des applications.

C++¶

Qt est construit en utilisant le langage C++, même si d’autres bindings existent. Le framework support un certain nombre de compilateurs, du GNU Compiler Collection (GCC), à la suite Virtual Studio. C’est donc un framework qui peut être utilisé pour créer des applications sur de nombreux systèmes d’exploitations (multi-plateforme).

Nokia¶

Nokia Corporation, la société qui contrôle activement le développement de Qt, est une entreprise de communication globale basée en Finlande. Nokia a beaucoup de filiales très connues, l’une d’elles étant Navteq.
